Формат PDF файла

В форму ниже необходимо загрузить pdf файл с отсканированными тестами. Ниже представлен список требований к входящему файлу:

1. Формат файла: PDF

2. Размер А4. Рекомендованное разрешение при сканировании >400dpi. Верх PDF должен быть сверху, т.е. все тесты должны быть правильно ориентированы.

3. Название файла должно быть на английском языке.

4. Максимальный размер файла 16 МБ.

5. Дождитесь подтверждения об отправке вверху экрана.

Шаблоны тестов для печати

В оба шаблона включена специальная область снизу и сверху экрана, которую можно редактировать и вставлять произвольный текст. Не допускайте попадания текста в область черных квадратов по углам текста.

Шаблон для математического теста (30 вопросов, выбран МАТ): mat-pdf-30q.pdf

Шаблон для общеобразовательного теста (40 вопросов, выбран ОТ): ot-pdf-40q.pdf


Шаблоны старых excel-шаблонов можно найти на Гитхабе. Они нужны только если основные pdf шаблоны по какой-то причине оказались нерабочими.
